🎬 AsilMedia для Lampa

Плагин для просмотра фильмов с asilmedia.org прямо в Lampa

Plugin Lampa Asilmedia

Шаг 1 — Установка плагина

Откройте Lampa → Настройки → Плагины → Добавить плагин и вставьте URL:

https://lampa-asilmedia.vercel.app/plugin.js

Шаг 2 — Выбор источника

В главном меню Lampa выберите источник AsilMedia.

Шаг 3 — Поиск и просмотр

Используйте поиск или выберите категорию:
Tarjima Kinolar · Seriallar · Multfilmlar · Yangilar

API: /api/index.js?action=search&query=...
